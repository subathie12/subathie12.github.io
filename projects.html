Portfolio Creation

This portfolio was brought to life through two main steps; Design and Build.

Through the design step, wireframes, storyboarding and visual display Figma and Canva were used to understand layout, typography, and spacing, and to generate ideas on how I could create my portfolio.

GitHub was used to build and deploy my portfolio using HTML/CSS and GitHub Pages. Adapting code and tools were used to achieve the functionality and design I envisioned, which is what you see today.
